# desafio-tecnico-bb

Solução desenvolvida em Excel para tratamento dos dados apresentados no desafio, aplicação das regras de validação e cálculo dos resultados por carteira e da agência.

## Solução

A solução foi desenvolvida utilizando fórmulas e recursos nativos do Excel, com foco em simplicidade, praticidade e fácil interpretação dos dados.

Os dados foram tratados e validados antes da realização dos cálculos, respeitando as regras definidas no desafio.

## Regras Aplicadas

- Validação da faixa de notas
- Validação de carteiras elegíveis
- Limite de duas avaliações por cliente e funcionário
- Identificação das avaliações válidas para apuração
- Cálculo do realizado por carteira
- Cálculo do realizado da agência
